ScheduleCollege — финальная памятка для защиты

Назначение:
Информационная система колледжа для составления, проверки, публикации и экспорта расписания учебных занятий.

Стек:
C#, .NET 8, ASP.NET Core Razor Pages, Entity Framework Core, SQLite, PostgreSQL/MySQL/MariaDB, Bootstrap, WinForms Tray.

Основные возможности:
1. Авторизация по логину и паролю.
2. Роли пользователей: Admin, Dispatcher, Teacher, Student.
3. Справочники: преподаватели, группы, дисциплины, аудитории, временные слоты.
4. Создание и редактирование расписания.
5. Проверка конфликтов по группе, преподавателю и аудитории.
6. Просмотр расписания с фильтрами.
7. Экспорт CSV, PDF и PNG.
8. Журнал действий пользователей.
9. Резервное копирование базы.
10. Переключение SQLite / PostgreSQL / MySQL / MariaDB через страницу администратора.
11. Доступ с других ПК локальной сети.
12. Подготовка внешнего доступа через Caddy/Nginx и домен.

Тестовые пользователи:
admin / admin123
Администратор, полный доступ.

dispatcher / dispatcher123
Диспетчер, работа со справочниками и расписанием.

teacher / teacher123
Преподаватель, просмотр данных.

student / student123
Студент, просмотр расписания.

Запуск через Visual Studio:
1. Открыть ScheduleCollege.sln.
2. Дождаться восстановления NuGet-пакетов.
3. Запустить ScheduleCollege.Desktop для запуска с иконкой в трее.
4. Либо запустить ScheduleCollege.Web напрямую.
5. Открыть http://localhost:5000.

Доступ в локальной сети:
1. Запустить приложение на основном ПК.
2. Открыть Администрирование -> Доступ.
3. Посмотреть адрес вида http://192.168.x.x:5000.
4. При необходимости открыть порт 5000 в брандмауэре Windows.

Примечание по простоте кода:
Проект специально сделан без Blazor, Aspire, Clean Architecture, CQRS и лишних абстракций. Основной упор сделан на понятный учебный код, который можно объяснить на защите.
