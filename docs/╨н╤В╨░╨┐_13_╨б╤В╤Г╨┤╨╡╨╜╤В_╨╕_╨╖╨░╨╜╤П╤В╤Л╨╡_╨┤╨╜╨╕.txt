Этап 13. Упрощение роли студента и занятые дни преподавателя

Изменения:
1. Для роли «Студент» в меню оставлены только две страницы:
   - Моё расписание;
   - Ближайшие экзамены.

2. Студент больше не видит общую панель управления, отчёты, справочники и админские разделы.
   При открытии главной страницы студент автоматически перенаправляется на своё расписание.

3. Для пользователей-студентов добавлено поле GroupCode — код учебной группы.
   По этому коду студент видит только расписание своей группы.

4. Добавлена страница «Ближайшие экзамены».
   Экзамены и зачёты определяются по названию дисциплины:
   - «... (экзамен)»;
   - «... (зачёт)».

5. Для роли «Преподаватель» добавлена страница «Мои занятые дни».
   Преподаватель может сам отметить дни недели, когда не может вести пары.

6. При составлении расписания уже занятые или недоступные преподаватели скрываются из списка выбора.
