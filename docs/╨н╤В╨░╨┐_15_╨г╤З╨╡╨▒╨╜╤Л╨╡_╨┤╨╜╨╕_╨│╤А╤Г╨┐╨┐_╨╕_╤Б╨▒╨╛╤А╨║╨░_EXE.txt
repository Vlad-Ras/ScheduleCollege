Этап 15. Учебные дни групп и сборка в один EXE

Что добавлено:
1. Добавлена таблица GroupStudyDays.
2. Для каждой группы и даты можно задать:
   - начальную пару;
   - последнюю пару;
   - выходной день;
   - комментарий.
3. Добавлена страница Расписание -> Учебные дни.
4. На странице можно:
   - заполнить неделю для одной группы;
   - заполнить неделю сразу для нескольких групп;
   - применить ко всем группам;
   - использовать шаблоны 1-6, 1-4, 3-6, 4-6;
   - скопировать настройки с прошлой недели.
5. При добавлении и редактировании занятия список пар теперь зависит от выбранной группы и даты.
6. Если группа в этот день учится со 2 пары, то 1 пара скрывается.
7. Если группа в этот день учится во вторую смену, доступны только поздние пары.
8. Если для группы указан выходной, занятия на этот день добавить нельзя.
9. Web-проект переведён на net8.0-windows и теперь может сам показывать иконку в трее.
10. Для запуска без трея можно передать параметр --no-tray.

Как собрать один EXE:
1. Открыть PowerShell в папке решения.
2. Выполнить:

dotnet restore

dotnet publish .\ScheduleCollege.Web\ScheduleCollege.Web.csproj -c Release -r win-x64 --self-contained true /p:PublishSingleFile=true /p:IncludeAllContentForSelfExtract=true /p:EnableCompressionInSingleFile=true

3. Готовый EXE будет здесь:
ScheduleCollege.Web\bin\Release\net8.0-windows\win-x64\publish\ScheduleCollege.Web.exe

4. Его можно переименовать в:
ScheduleCollege.exe

Примечание:
- Один EXE запускает веб-сервер и иконку в трее.
- Адрес системы: http://localhost:5000
- Для запуска без иконки трея:
ScheduleCollege.exe --no-tray
